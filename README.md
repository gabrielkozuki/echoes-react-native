# Echoes

Echoes is a gaming memory journal — a space to record what each game left in you, and revisit those memories unexpectedly.

## Stack

- **React Native** with Expo (Dev Client) + TypeScript
- **@shopify/react-native-skia** — Constellation screen canvas
- **expo-sqlite** — local database, no backend
- **Zustand** — global state
- **RAWG API** — game search

## Getting started

```bash
npm install
npx expo run:android
# or
npx expo run:ios
```
